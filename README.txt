This adressbook app is written in PHP and HTML and uses sqlite as a persistent database.

Requirements:
	* PHP
	* sqlite

Please make sure the extension 'pdo_sqlite' in php.ini is uncommented/activated.

Usage:
To add a new contact/adress use the form on the top of the page.
The contacts are read from the DB and displayed in the table below.
To edit an adress click the 'edit' button.
To submit your changes click the submit button in edited row.
To delete an adress click the 'delete' button.
The contacts can be sorted right above the table.
